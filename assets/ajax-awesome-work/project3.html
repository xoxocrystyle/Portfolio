Student Grade Table
by Crystal Navarro
Student Grade Table was my first project where I used AJAX calls and Bootstrap 4. It uses the the basic functionality of persistant storage: CRUD. What was a simple one page app using CRUD, is now a web application where the user can login and is given multiple functionalities making the UI/UX more useful. The user, generally a teacher, signs in via a local login and is allowed to enter a student's name, course, and grade. Once entered, the overall grade average is calculated. The user us allowed to add, delete, sort, and change information.

Technologies used: HTML5, CSS3, Bootstrap 4, jQuery, PHP, MySQL
